# PanoramaLite AR 光学原型演示

本仓库提供一个基于 Web 的 AR 光学方案原型页面（`index.html`），展示大视场、小体积、低重量的近眼显示设计思路，并包含可交互的参数沙盘。

## 使用方式
1. 打开 `index.html` 即可在浏览器中查看演示，无需构建或依赖。
2. 页面提供核心指标、方案架构、人机交互逻辑以及可制造性分析，支持滑杆快速估算重量、成本和良率。
3. 可直接通过浏览器的打印功能将页面导出为 PDF，便于提交说明书或 PPT。

## 核心参数示例
- 视场：92° × 82°（对角）
- 模组 TTL：13.6 mm
- 模组重量：18 g
- MTF@20 lp/mm：≥ 0.35
- Pupil swim：< 1%

## 目录结构
- `index.html`：单页原型，包含样式、交互逻辑及方案说明。

如需扩展，可在同目录下添加图片、仿真结果或额外文档并在页面中引用。
